# Attribution

The Trinity Demonstrator builds on the work of many people in the Prolog
community. This document acknowledges contributions whose code, ideas, or
infrastructure made this project possible.

## Core platforms

- **Jan Wielemaker** — author and maintainer of [SWI-Prolog](https://www.swi-prolog.org/),
  the Prolog system on which the server-side of the Trinity Demonstrator runs.
  SWI-Prolog provides the engine, the standard libraries, the HTTP and WebSocket stacks, and
  the other wonderful infrastructure that this project depends on.

## Libraries and example programs

- **Richard O'Keefe** — for various libraries and example programs from
  *The Craft of Prolog* and the wider Prolog literature, several of which are
  adapted or referenced in the demonstrator's example collection.
